    RingCentral for Microsoft Teams

    Embed reliable and powerful calling into Microsoft Teams

    Use the RingCentral embedded app, presence sync, and Admin Portal tools so your organization can call, message, and stay in sync—without leaving Teams.

    View on App Gallery General Questions

    Install the embedded app (Admin Portal)

    Admins can remotely install the RingCentral for Microsoft Teams embedded app using OAuth or Microsoft Azure. Choose a path, then open the Admin Guide for full steps.

    • OAuth

      Use Connect via OAuth in the Admin Portal when you prefer the browser consent flow over Azure app registration.

      OAuth in Admin Guide

    • :simple-microsoftazure:{ .lg .middle } Microsoft Azure

      Register an app in Azure AD, grant Microsoft Graph permissions, and connect credentials in the RingCentral Admin Portal.

      Azure setup in Admin Guide
