# RingCentral Actions in Zapier

RingCentral supports various actions that you can use in your workflows.

### Example: Sending an SMS When a New Email Arrives

1. Select Gmail as the trigger app.
2. Choose New Email as the trigger event.
3. Connect your Gmail account.
4. Set up a filter to select the relevant emails.
5. Select RingCentral as the action app.
6. Choose Send SMS/MMS as the action event.
7. Map the email subject to the SMS text field.
8. Test and publish the Zap.

## Other RingCentral Actions

* Create a Video Meeting: Automatically schedule a meeting.
* Send Fax: Send documents via fax.
* Generate RingOut Call: Connect two numbers instantly.
