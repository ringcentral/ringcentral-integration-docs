# RingCentral for Salesforce  

RingCentral for Salesforce integrates your Salesforce.com account with your RingCentral services, creating a seamless communication experience. This integration enhances customer retention, boosts agent productivity, and streamlines business processes for improved performance.

## System Requirements

### Salesforce Edition:
- Professional, Enterprise, or Unlimited

### RingCentral Subscription:
- MVP Premium or higher

### Hardware Requirements (For optimal Salesforce performance):
- Octane 2.0 score of 30,000 or higher  
- Network latency of 150 ms or less  
- Download speed of 3 Mbps or greater  
- Minimum of 8 GB of RAM, with at least 3 GB available for Salesforce browser tabs

### Recommended Operating System and Software:
- Latest versions of Windows or macOS (32-bit or 64-bit)  
- Current versions of Chrome, Firefox, Edge Chromium, or Safari  
- Latest RingCentral app (optional)  
- Latest RingCentral Phone app (optional)

### Minimum Operating System and Software:
- Windows XP SP2  
- macOS X 10.8  
- Chrome 56, Firefox 52, Safari 6.0.5

## About This Guide  
This user guide is tailored for end-users of RingCentral for Salesforce. It provides instructions on using the application, highlights known issues and limitations, and offers basic troubleshooting tips. It is not intended for system or network administrators and does not cover installation or configuration processes for either the Salesforce.com instance or user desktops.

This guide specifically supports RingCentral for Salesforce version 6.18.x and applies to Salesforce Lightning. For documentation on previous versions or related guides, please visit the Salesforce AppExchange.
