# :material-archive-arrow-up: RingCentral Archiver

**Securely archive your business communications to the cloud**

RingCentral Archiver enables users to automatically archive their RingCentral Call data to external storage locations, ensuring compliance and easy data retrieval.

[Get Started :material-arrow-right:](enable-archiver.md){ .md-button .md-button--primary }
[View Storage Options :material-cloud-outline:](connect-storage.md){ .md-button }

</div>

## Supported Data Types

Archive all your critical communication data in one place.

<div class="grid cards" markdown>

-   :material-phone-voip:{ .lg .middle } **Call Recordings**

    ---

    Automatically archive all inbound and outbound call recordings for compliance and training.

-   :material-voicemail:{ .lg .middle } **Voicemails**

    ---

    Preserve voicemail messages with full metadata and transcription support.

-   :material-fax:{ .lg .middle } **Fax**

    ---

    Archive sent and received faxes with complete transmission details.

-   :material-message-text:{ .lg .middle } **SMS**

    ---

    Store SMS conversations for record-keeping and compliance requirements.

-   :material-forum:{ .lg .middle } **Team Messages**

    ---

    Archive team messaging conversations and shared content.

</div>
